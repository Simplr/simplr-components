# Simplr Icon & Icon Button

### Usage

##### Import
```js
// Icon
import 'simplr-components/icon/icon.js';

// Icon button
import 'simplr-components/icon/icon-button.js';
```

##### Use
```html
<!-- icon -->
<simplr-icon></simplr-icon>

<!-- icon button -->
<simplr-icon-button></simplr-icon-button>
```
