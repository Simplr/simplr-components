# Simplr Select

### Usage

```js
import '@simplr-wc/select';

const options = [
    {label: "Banana", value: 1},
    {label: "Strawberry", value: 2},
    {label: "Kiwi", value: 3},
    {label: "Peach", value: 4},
];
```

```html
<simplr-select
    label="Simple select"
    .options=${options}
></simplr-select>
```
