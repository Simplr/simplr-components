# Simplr Radio Group

### Usage

```js
  import 'simplr-components/input/radio-group/radio-group.js';

  const radioSelections = [
    { label: 'Option 1', value: 1 },
    { label: 'Option 2', value: 2 },
    { label: 'Option 3', value: 3 },
    { label: 'Option 4', value: 4 },
  ];
```

```html
    <simplr-radio-group
      name="inputs-one"
      .options="${radioSelections}"
      .selectedValue=${2}
    ></simplr-radio-group>
```
