# Simplr Top App Bar with Action Items

### Usage

```js
import 'simplr-components/top-app-bar/top-app-bar.js';

const actionItems = [
  {icon: "code", onclick: () => alert("Code action")},
  {icon: "add", onclick: () => alert("Add action")}
];
```

```html
<simplr-top-app-bar
  .actionItems=${actionItems}
></simplr-top-app-bar>
```
